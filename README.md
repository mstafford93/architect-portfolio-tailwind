Architect-Portfolio-Tailwind

Design based on example template from Itonlinelearning (where I am learning to code).
Layout to strictly avoid "position:absolute". With the main layout to be showcasing Flexbox and Grid.

Dependencies:
  - HTML 5
  - Tailwind CSS (v3.4.17)

Usage
  - Run in a source-code text editor (e.g. Visual Studio Code) to view code
  - Run in web browser to see website

Configeration 
  - No config needed

Contributing Guidelines
  - Project is to test my understanding of Tailwind CSS, by only using Tailwind.
